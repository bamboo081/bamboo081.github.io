title: "How the network side of the homelab evolved"
---

The network side of this homelab started getting useful once the lab stopped being just another part of the home LAN.

Key changes:

- reusing an older FiOS router
- separating the lab from the main home network flow
- using the laptop and desktop in a more structured admin-and-server relationship

That separation made it much easier to think about future services like DNS filtering, dashboards, reverse proxies, and remote access in a way that would not interfere with the rest of the house.
