My homelab currently revolves around three main pieces:

- a dedicated FiOS router for the lab
- my Lenovo Legion laptop for access and testing
- my desktop as the main homelab machine

The biggest improvement so far has been shifting from a loose collection of devices into a more intentional lab setup with its own router path, Ubuntu as the main server direction, and a growing container-based stack.

The biggest active work right now is reliability:

- making DNS filtering work cleanly
- improving remote access quality
- reducing service conflicts
- preparing for better monitoring and self-hosted applications
