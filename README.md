# Bamboo Homelab GitHub Pages Site

This repository powers my GitHub Pages site for documenting my homelab build, setup progress, services, and troubleshooting.

## What is included

- `index.md` — homepage and current status
- `timeline.md` — chronological summary of what has been done so far
- `services.md` — services, experiments, and planned additions
- `troubleshooting.md` — issues I hit and what I learned from them
- `_posts/` — blog-style entries for progress updates
- `_config.yml` — GitHub Pages / Jekyll configuration
- `assets/css/style.scss` — small style overrides for a cleaner blog-like layout

## How to use

1. Open your GitHub Pages repository.
2. Delete or replace the default placeholder files.
3. Upload everything from this folder to the root of the repository.
4. Commit the changes.
5. In GitHub Pages settings, make sure the site is publishing from the correct branch.

## Notes

This version removes sensitive details such as private IP addressing, local hostnames, and other network-specific identifiers.
